# MostlySports

MostlySports is a fan-built wiki project for the Mostly Sports YouTube show from Barstool Sports.

The site is starting with the Athlete Name Hall of Fame: a collection of athletes with memorable, unusual, or perfectly show-worthy names. Each entry includes the athlete's name, sport, team, photo, and a short tenure note.

Over time, the goal is to grow this into a broader Mostly Sports reference site, with more show-related pages, recurring bits, lists, segments, and other wiki-style material.

## Current Starting Point

The first page is a static HTML version of the Athlete Name Hall of Fame. It currently includes:

- A main hall of fame section
- An honorable mentions section
- Athlete cards with photos and basic details
- Inline styling in `index.html`

## How to View

Open `index.html` in a web browser. No build step, server, or package installation is required.
