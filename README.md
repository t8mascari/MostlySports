# Mostly Wiki

Mostly Wiki is a fan-built wiki project for the Mostly Sports YouTube show from Barstool Sports.

The site is starting with the Name Hall of Fame: a collection of athletes with memorable, unusual, or perfectly show-worthy names. Each entry includes the athlete's name, sport, team, photo, and a short tenure note.

Over time, the goal is to grow this into a broader Mostly Sports reference site, with more show-related pages, recurring bits, lists, segments, and other wiki-style material.

## Current Site

The current site is a static HTML page with tabbed sections. It includes:

- A `Name Hall of Fame` tab
- A `Jersday History` tab with a coming-soon `Jersday Database` placeholder
- An honorable mentions section
- Athlete cards with photos and basic details
- A local Mostly Wiki icon used for the header logo and browser favicon
- A `Support the Organizer` Venmo link in the footer
- Inline styling in `index.html`

## How to View

Open `index.html` in a web browser. No build step, server, or package installation is required.

## Project Files

- `index.html`: Main static page for the wiki.
- `assets/mostly-wiki-icon.png`: Local logo and favicon image.
- `AGENTS.md`: Project guidance for Codex.
